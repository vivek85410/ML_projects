# ML_projects
Project 1 : Algerian forest fire project -------------------------------------------

The dataset includes 244 instances that regroup a data of two regions of Algeria, namely the Bejaia region located in the northeast of Algeria and the Sidi Bel-abbes region located in the northwest of Algeria.

122 instances for each region.

The period from June 2012 to September 2012.
The dataset includes 11 attributes and 1 output attribute (class)
The 244 instances have been classified into fire (138 classes) and not fire (106 classes) classes.

columns : 
day
month
year - Year which is 2012
Temperature - temperature noon (temperature max) in Celsius degrees: 22 to 42
RH - Relative Humidity in %: 21 to 90
Ws - Wind speed in km/h: 6 to 29
Rain - total day in mm: 0 to 16.8
FFMC - Fine Fuel Moisture Code (FFMC) index from the FWI system: 28.6 to 92.5
DMC - Duff Moisture Code (DMC) index from the FWI system: 1.1 to 65.9
DC - Drought Code (DC) index from the FWI system: 7 to 220.4
FWI (prediction) - FWI stands for Fire Weather Index, a key numerical rating used in the Algerian forest fire dataset to indicate wildfire danger levels. It measures the potential intensity of a forest fire based on meteorological conditions—such as temperature, humidity, wind speed, and rain—and indicates how easily a fire can ignite and spread. 
The dataset focuses on the Bejaia and Sidi Bel-abbes regions, analyzing data from June to September 2012 to predict the risk of fire and non-fire incidents. 

Higher FWI values indicate a higher probability of forest fire, aiding authorities in resource allocation and early warning systems.
